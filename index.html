Projetos

Os projetos apresentados a seguir são estudos de caso desenvolvidos a partir de dados públicos, utilizando marcas fictícias, com foco em análise de dados, desempenho e tomada de decisão estratégica.

Projeto OfficeMart

Projeto em Excel para análise de desempenho de vendas da OfficeMart, utilizando análise de Cohort e segmentação RFM para entender retenção de clientes, identificar perfis de consumo e avaliar o desempenho de produtos e lojas, gerando insights para melhorar estratégias comerciais e aumentar o faturamento.

Projeto EasyRent

Projeto de análise de acomodações da EasyRent desenvolvido em dashboard analítico, permitindo acompanhar crescimento da oferta, distribuição geográfica e variação de preços, gerando insights para apoiar decisões estratégicas de expansão e posicionamento no mercado de locações.

Projeto Mesa Livre

Dashboard desenvolvido em Python para análise de dados do marketplace Mesa Livre, permitindo compreender o desempenho dos restaurantes, padrões de oferta e comportamento do mercado, gerando insights estratégicos para apoiar decisões e impulsionar o crescimento da plataforma.

Projeto Aroma Café

Projeto em Google Sheets utilizando regressão linear no Xminer para analisar o desempenho da rede Aroma Café, combinando análises descritiva, diagnóstica e preditiva para entender fatores que impactam o faturamento e estimar receitas futuras, apoiando decisões estratégicas do negócio.
